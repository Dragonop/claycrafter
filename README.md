# claycrafter
Minetest clay crafter mod. Adds a way to craft clay out of compressed dirt.

Dependencies: 
Default
Vessels
Bucket

claycrafter.lua is derivated from furnace.lua (minetest_game) by PilzAdam and Amaz1. 
Edited by everamzah to work for this mod.
